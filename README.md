
# Ideaslist

A curated list of awesome projects ideas developed in Ruby on Rails.

## Usage

1. Clone the repo  
  ```git clone https://github.com/shadonet/ideaslist```

2. Install dependencies  
  ```bundle install```

3. Create database with rake.
    ```
    rake db:create
    rake db:migrate
    ```
